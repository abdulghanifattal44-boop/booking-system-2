# Роли команды (Team Roles)

- **PM**: план, Kanban, распределение задач, PR review, демо/защита
- **Analyst**: требования, use cases, business rules
- **DB**: ERD, миграции, ограничения, seed
- **Backend**: API, JWT, booking logic, Swagger/OpenAPI
- **Frontend**: Customer/Admin UI, интеграция API, обработка ошибок
- **DevOps**: Docker Compose, env, CI (опционально)
